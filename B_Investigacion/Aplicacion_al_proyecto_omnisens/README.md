# Aplicación al Proyecto – OmniSens Industrial Suite

## Descripción

Esta carpeta reúne la documentación relacionada con la aplicación práctica de los conceptos desarrollados durante el proyecto OmniSens Industrial Suite.

Su objetivo es evidenciar pruebas, validaciones, comportamientos operativos y resultados obtenidos durante la implementación de la solución IoT.

---

## Documentos incluidos

### 📄 resiliencia_operativa.md

Documenta la validación de resiliencia operativa del sistema ante fallas de conectividad.

Contenido:

- Simulación de caída de red.
- Verificación del procesamiento local.
- Validación de actuadores.
- Continuidad operativa.
- Comportamiento sin acceso al servidor.

Objetivo:

Comprobar que el sistema mantiene funciones críticas aun sin conectividad con servicios remotos.

Resultado esperado:

Garantizar la continuidad de monitoreo y control mediante procesamiento local.
